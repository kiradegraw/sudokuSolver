- sudoku.py implements a sudoku game solver with backtracking (naive and smart)
- Implemented in Python3. No parameters are needed for this program.
- Program will ask user for algorithm to be used (naive or smart) after running program
	- when prompted: enter '1' for naive and '2' for smart
- Program displays initial and solution board, as well as the time taken to solve
- The provided text file: "sudokuPuzzle.txt" holds the initial puzzle. Change the text file to run
	the program with a different initial puzzle.


- Run: python3 sudoku.py
